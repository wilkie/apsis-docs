Apsis:: Namespace Engine

Classes

  • Input - Handles which keys are pressed and can be polled to determine when an Action is still held.
  • Object - The object engine manages paths and loaded objects within the system and coordinates on your behalf with the object and name registry. Load and query all system objects through this interface.
  • Scene - This class manages the current scene.
  • System - This engine is the root of all resources.
  • Event - Contains an input event.
