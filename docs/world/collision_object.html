Apsis::World:: Class CollisionObject

Source

world/collision_object.h

Methods

  • objectType - Returns the type of collision object.
  • collisionType - Returns the type of collision that occured.
  • direction - Returns the direction the colliding object is going to hit this object.
  • redirection - Returns the redirection vector that should be followed as a result of this collision.
  • edge - Returns the edge that was hit upon the collision.
  • point - Returns the point that was hit upon the collision.
  • repositionPoint - Returns the point the object that hit should be positioned in order to rest touching the collision object.
  • actor - Returns the Actor object that resulted in this collision.
  • actor - Sets this collision object to reflect the given Actor.
  • tile - Returns the Tile object that resulted in this collision.
  • tile - Sets this collision object to reflect the given Tile.
  • map - Returns the Map object that resulted in this collision.
  • map - Sets this collision object to reflect the given Map.
  • collideEvent - Returns the collided_with_* event that corresponds to this object.
