Apsis::World:: Class Actor

Source

world/actor.h

Methods

  • sheet - Return: the Apsis::Sprite::Sheet for the Apsis::Sprite.
  • position - Return: the Apsis::Geometry::Rectangle for the Apsis::Sprite.
  • animate - Sets the current animation to be played by this Apsis::Sprite.
  • nextFrame - Advances the frame to the next sprite.
  • update - Updates the current time for the Sprite. Affects animations and movements.
  • act - Reacts to an input action.
  • collide - Responds to collisions.
  • attachRule - Adds the given rule to the actor's rule set.
  • draw - Draws the actor.
  • name - Returns the name of this Actor.
  • respond - Responds to events.
  • object - Returns an immutable reference to the Actor's World::Object.
