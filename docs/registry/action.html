Apsis::Registry:: Class Action

Source

registry/action.h

Methods

  • load - Loads a JSON bindings description that will load default bindings for a set of actions.
  • id - Registers or returns the action for the given name.
  • id - Returns the Action by the given index.
  • count - Returns the number of actions in the system.
  • yield - Yields the action triggered by the given key binding. Returns true when a binding yields an event. In this case, action_id will be set to the id of the action.
  • bindPrimary - Binds the given key binding as the primary means of activating this action.
  • bindSecondary - Binds the given key binding as the secondary means of activating this action.
  • primary - Returns the primary key binding for this action.
  • secondary - Returns the secondary key binding for this action. If there is no primary, this will actually set the primary binding.
  • bindingCount - Returns the number of bindings that will trigger this action.
  • clearPrimary - Clears the primary binding for this action.
  • clearSecondary - Clears the secondary binding for this action.
  • clear - Clears all bindings for this action.
  • id - Returns the unique identifier for this Action within the system.
