Apsis::Registry:: Class Rule

Source

registry/rule.h

Methods

  • load - Loads a Rule from the rule info represented by the given file.
  • id - Returns the unique id for this rule.
  • name - Returns the name of this rule.
  • path - Returns the path of the rule description, if used. Will return "" when such a description was not used.
  • id - Registers or returns the existing id for the given property.
  • supercedes - Returns true when the given rule, specified by name, is superceded by this rule. Returns false otherwise.
  • ruleCount - Returns the number of subrules.
  • rule - Returns the subrule at the given index.
  • collideFunctionCount - Returns the number of collide functions.
  • collideFunction - Returns the collide function at the given index.
  • updateFunctionCount - Returns the number of update functions.
  • updateFunction - Returns the update function at the given index.
  • replacement - Returns a name of rules that this rule replaces when the rule actively updates with the given index.
  • replacementCount - Returns the number of names of rules that this rule will replace when the rule actively updates.
