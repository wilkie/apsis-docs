Apsis::Interface:: Class Window

Source

interface/window.h

Methods

  • position - Returns the position of this Window within its parent.
  • next - Returns the next sibling Window.
  • prev - Returns the previous sibling Window.
  • child - Returns the first child Window.
  • childCount - Returns the number of children.
  • add - Adds the given Window as a child.
  • detach - Detaches this window from its parent.
  • attached - Returns true if this Window has a parent.
  • draw - Draws this Window and all of its children.
