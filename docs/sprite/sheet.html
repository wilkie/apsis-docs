    enumerateSprites

    Returns: -1 when no more sprites match, or the index of the next available sprite that matches the wildcard given by name. For instance wildcard = "down_*" will yield the next sprite after or including the index at last that begins with "down_". Use last = 0 to start, and then pass the return value + 1 in the next invocation.
