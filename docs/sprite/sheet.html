Methods

  • load -
  • texture - Returns: The Texture for this SpriteSheet.
  • textureCoordinates - Fills the given coordinate array with the texture coordinates of the given sprite. The sprite index is assigned by the order in which sprites are listed in the description file.
  • textureCoordinates - Fills the given coordinate array with the texture coordinates of the given sprite. The sprite is located by comparing the name given with the one described by the description file. Returns: true when the sprite is found, false otherwise.
  • count - Returns: the number of sprites contained in the sheet.
  • width - Returns the width of the sprite at the given index.
  • height - Returns the height of the sprite at the given index.
  • id - Returns the id of this sprite sheet in the system.
  • enumerateSprites - Returns: -1 when no more sprites match, or the index of the next available sprite that matches the wildcard given by name. For instance wildcard = "down_*" will yield the next sprite after or including the index at last that begins with "down_". Use last = 0 to start, and then pass the return value + 1 in the next invocation.
  • draw - Draws the given sprite at the given location.
