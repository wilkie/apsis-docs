Apsis::Sprite:: Class Thing

Source

sprite/thing.h

Methods

  • load - Loads a Thing from the thing info represented by the given file.
  • get - Retrieves an already loaded Thing of the given id.
  • sheet - Returns the sprite sheet associated with this Thing.
  • animation - Returns the animation represented by the given name.
  • animationId - Returns the id of the animation represented by the given name.
  • animationById - Returns the animation represented by the given id.
  • animationCount - Returns the number of animations known to this Thing.
  • id - Returns the unique id of this Thing.
  • object - Returns the Object representation of this Thing.
  • rules - Returns the set of rules attached to this Thing.
  • name - Returns the name of this thing.
