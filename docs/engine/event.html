Apsis::Engine:: Class Event

Source

engine/event.html

Methods

  • type - The type of event.
  • systemEvent - Returns the system event identifier.
  • systemEvent - Sets the system event. Also will set type to refer to such an event.
  • binding - Returns the key binding associated with this event.
  • binding - Sets the event data to that of the given key binding. Will also set type to indicate the event is a key binding depending on whether it has been indicated that the key was pressed or released.
