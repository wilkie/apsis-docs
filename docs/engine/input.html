Apsis::Engine:: Class Input

Source

engine/input.html

Methods

  • basic - Constructs a bare input engine.
  • press - Tells the input engine to press the given key. It will return true when an action is triggered.
  • release - Tells the input engine to release the given key. It will return true when an action is triggered.
  • isActionHeld - Returns true when the given action is still pressed.
