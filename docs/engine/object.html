Apsis::Engine:: Class Object

Source

engine/object.h

Methods

  • load - Constructs an Object engine from the given JSON description. Should point to a file containing a "paths" type.
  • load - Constructs an Object engine from the given JSON object. Should be an object referred to by the name "paths".
  • basic - Constructs an Object engine that has default properties.
  • loadThing - Loads or returns the existing Thing object with the given name or path.
  • loadMap - Loads or returns the existing Map object with the given name or path.
  • loadRule - Loads or returns the existing Rule object with the given name or path.
  • loadScene - Loads or returns the existing Scene object with the given name or path.
  • loadBindings - Loads bindings.
