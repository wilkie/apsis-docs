Apsis:: Class KeyBindingRepository

Source

input/key_binding_repository.h

Methods

  • registerEvent - Creates a new binding entry with the specified name and the given defaults. Use NULL to specify no initial mapping. The parameter event will be returned when a key is translated.
  • rebindPrimary - Rebinds a key combination to the primary binding of the given event. The event must have been registered by registerEvent().
  • rebindSecondary - Rebinds a key combination to the secondary binding of the given event. The event must have been registered by registerEvent().
  • yieldEvent - Determines the event identifier that corresponds to the given key binding. Returns true if the binding was resolved.
