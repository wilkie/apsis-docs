Apsis:: KeyBindingRepository

Source

input/key_binding_repository.h

Methods

  • Name

    registerEvent

    Description

    Creates a new binding entry with the specified name and the given defaults. Use NULL to specify no initial mapping. The parameter event will be returned when a key is translated.

    Parameters

    • name
    • event
    • primary
    • secondary

  • Name

    rebindPrimary

    Description

    Rebinds a key combination to the primary binding of the given event. The event must have been registered by registerEvent().

    Parameters

    • name
    • primary

  • Name

    rebindSecondary

    Description

    Rebinds a key combination to the secondary binding of the given event. The event must have been registered by registerEvent().

    Parameters

    • name
    • secondary

  • Name

    yieldEvent

    Description

    Determines the event identifier that corresponds to the given key binding. Returns true if the binding was resolved.

    Parameters

    • binding
    • event
